VEXIS-CLI-1 Released

High-precision AI automation tool evolved from GUI to CLI operation

We are pleased to announce the release of VEXIS-CLI-1. This is a VEXIS that has undergone significant code modifications to transform VEXIS-1.1, which operates GUI, into an AI for CLI operation.

🎯 Improved Accuracy through CLI Conversion

The conventional standard VEXIS, which was neither CLI nor anything special, had very low accuracy. However, VEXIS-CLI-1 has significantly improved accuracy. Since the base model is a language model, CLI operation through text was indeed more suitable than GUI.

🤖 Dramatic Expansion of Model Options

With the shift to text-based tasks where LLMs excel, conventional VEXIS could only use the Gemini 3 family, which excels in visual reasoning. However, CLI conversion has dramatically expanded model options, making it possible to comfortably use even relatively small OSS models.

�️ Cross-Platform Compatibility

VEXIS-CLI-1 runs not only on pure CLI Linux environments, but also on terminals and command prompts of desktop operating systems such as Windows and macOS. This allows more users to easily utilize AI automation tools in their own environments.

�� Future Development Direction

Even with the advent of VEXIS-CLI, we plan to continue evolving the conventional VEXIS-1.1 lineage. We will continue to provide optimal AI automation tools tailored to the characteristics of both GUI and CLI operations.

VEXIS-CLI-1 achieves higher accuracy and broader model compatibility by specializing in text-based operations. This has created an environment where more users can easily utilize AI automation tools.

Expanding possibilities with text-operated AI automation.
