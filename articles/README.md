# 記事の追加方法

このフォルダに新しい記事を追加するだけで、自動的に記事一覧に表示されるようになります。

## 新しい記事を追加する手順

### 1. 記事HTMLファイルを作成
`article-template.html` をコピーして、新しいファイル名で保存します。（例: `new-article.html`）

### 2. 記事内容を編集
コピーしたファイルのプレースホルダーを置き換えます：

- `{{TITLE}}` - 記事のタイトル
- `{{SUBTITLE}}` - 記事のサブタイトル
- `{{DATE}}` - 記事の日付（例: 2024年2月28日）
- `{{CATEGORY}}` - 記事のカテゴリ（例: 技術ブログ, プロジェクト, お知らせ）
- `{{INTRO}}` - 記事の導入文
- `{{CONTENT}}` - 記事の本文（HTML形式）

### 3. articles.json を更新
`articles.json` ファイルに新しい記事の情報を追加します：

```json
{
  "id": "new-article",
  "title": "新しい記事のタイトル",
  "subtitle": "記事のサブタイトル",
  "date": "2024年2月28日",
  "category": "技術ブログ",
  "excerpt": "記事の抜粋（記事一覧に表示される短い説明）",
  "tags": ["タグ1", "タグ2", "タグ3"],
  "url": "./articles/new-article.html"
}
```

### 4. アップロード
作成したHTMLファイルと更新した `articles.json` をサーバーにアップロードするだけです！

## 注意点

- 記事IDは一意のものにしてください（英数字とハイフンのみ使用）
- 日付形式は「YYYY年M月D日」に統一してください
- タグは3つまで推奨です
- 画像を使用する場合は `images` フォルダを作成して管理してください

## 自動化について

将来的には、`articles.json` の更新も自動化する予定です。現時点では手動更新が必要ですが、HTMLファイルの追加だけで記事を公開できるようになっています。
